Propuesta de Celda Robotizada

Se identificó que en el proceso de corte había un cuello de botella, ya que para fabricar diferentes elementos de los productos como el sinfín corona, los engranajes cónicos o los engranajes helicoidales, solo se tenía una cortadora manual que no era lo suficientemente eficiente. Además, este proceso de corte era bastante repetitivo.

Por eso se decidió formar una celda con estos elementos: un robot de la marca ABB, referencia IRB2600, una cortadora semiautomática y unas bandas planas que ayudarán a transportar la materia prima. La cortadora hará que el proceso sea más rápido y eficiente, y el robot ayudará con el almacenamiento.

Como complementos a estos elementos tenemos un gripper para el robot, referencia PHL 50 060 de la marca Schunk, el cual cumple con las características para sujetar las barras de diferentes dimensiones y diferentes pesos que podrían obtenerse en el proceso de corte.

También tenemos el sensor inductivo LJ12A3-4-Z/BX, el cual están dispuestos varios a lo largo de la banda transportadora. Tres de ellos configuran las longitudes para que la banda, al momento de detectar una barra, pause su movimiento, sujete el material de corte, efectúe el corte y luego lo deje seguir para que el robot continúe con su posterior almacenamiento.
