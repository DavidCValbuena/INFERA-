Implementación en NX de un sistema de estación de corte

Se desarrolló un sistema de automatización para una estación de corte de materia prima utilizando Digital Factory en NX. El proyecto integra tres plataformas principales:

  1. Studio 5000: Donde se programa la lógica de control del sistema
  2. Ignition: Servidor intermedio que conecta Studio 5000 con NX mediante protocolos OPC DA y OPC UA
  3. NX (Siemens): Donde se simula y visualiza la estación de trabajo

Componentes de la estación

  • Dos bandas transportadoras (entrada y salida)
  • Sistema de corte con disco y eje Z
  • Dos prensas/mordazas para fijar el material
  • Cuatro sensores (tres para longitudes de corte y uno para detección de salida)

Las barras de 3 metros entran por la banda transportadora, se detectan mediante sensores que determinan la longitud de corte deseada, se fijan con prensas, se cortan con el disco, y finalmente el material cortado sale por la banda de salida. El sistema permite seleccionar diferentes longitudes de corte preprogramadas y cuenta con funciones de arranque, parada y parada de emergencia. Las piezas cortadas serán posteriormente mecanizadas para fabricar componentes de motorreductores.
